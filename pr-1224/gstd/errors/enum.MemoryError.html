Expand description

Memory error.

Variants

OutOfBounds

The error occurs when a program tries to allocate more memory than allowed.

InvalidFree(u32)

The error occurs in attempt to free-up a memory page from static area or outside additionally allocated for this program.

MemoryAccessError

The error occurs in attempt to access memory page outside pages area allocated for this program.

InvalidPageDataSize(u64)

WASM page does not contain all necessary Gear pages.
